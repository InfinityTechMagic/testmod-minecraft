# TestMod for Minecraft 1.20.1

Мод для Minecraft 1.20.1 на Forge, добавляющий рубины в игру.

## Особенности
- Добавляет новый предмет: Ruby
- Простая установка
- Совместимость с Forge 47.2.0

## Установка
1. Скачайте файл `testmod-1.0.0.jar` из раздела Releases
2. Поместите в папку `mods` вашего Minecraft
3. Запустите игру

## Для разработчиков
Для сборки требуется:
- JDK 17
- Minecraft Forge 47.2.0
- IntelliJ IDEA
